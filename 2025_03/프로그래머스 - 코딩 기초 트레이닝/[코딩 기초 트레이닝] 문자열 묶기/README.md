### 문제 설명

### 제한 사항

- 1 ≤ strArr의 길이 ≤ 100,000
- 1 ≤ strArr의 원소의 길이 ≤ 30
- strArr의 원소들은 알파벳 소문자로 이루어진 문자열입니다.

### 입출력 예

| strArr                    | result |
| ------------------------- | ------ |
| ["a","bc","d","efg","hi"] | 2      |
